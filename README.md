# Маори Лукьяновка

Многостраничный сайт бронирования частного дома Маори в Лукьяновке.

## Railway

Проект запускается как обычный Node.js сервис:

- Build Command: пусто
- Start Command: `node server.js`
- Branch: `main`

## Переменные окружения

Для админ-панели обязательно задайте:

- `ADMIN_PASSWORD` - пароль для входа на `/admin`

Для Telegram-заявок можно задать:

- `TELEGRAM_BOT_TOKEN` - токен бота от BotFather
- `TELEGRAM_CHAT_ID` - id чата, куда отправлять заявки

Если Telegram-переменные не заданы, форма все равно работает на сайте и показывает подтверждение, но реальная доставка хозяйке будет отмечена как неподключенная.

## Админ-панель

Админка доступна по адресу:

- `/admin`

В ней можно редактировать тексты, фотографии и повторяемые элементы по разделам:

- главная
- пять комнат и общие зоны
- Катерина, питание и фестивали
- ретриты, группы и активности
- реальные отзывы со ссылками на источники
- бронь
- контакты, карта, оплата и политика

Контент хранится в `data/content.json`. Публичные страницы подтягивают его через `/api/content`, а админка сохраняет изменения через защищенный `/api/admin/content`.

Важно: Railway может пересоздавать файловую систему при редеплое. Поэтому изменения, сохраненные через админку на сервере, могут откатиться к версии из GitHub после нового деплоя. Перед редеплоем используйте кнопку "Экспорт" в админке и при необходимости обновляйте `data/content.json` в репозитории.

## Страницы

- `/` - главная
- `/rooms` - комнаты
- `/host` - Катерина, питание и фестивали
- `/location` - ретриты, экскурсии и активности
- `/booking` - бронирование
- `/admin` - админ-панель
