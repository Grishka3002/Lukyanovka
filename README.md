# Маори Лукьяновка

Многостраничный сайт частного дома Маори в Лукьяновке.

## Railway

Проект запускается как обычный Node.js сервис:

- Build Command: пусто
- Start Command: `node server.js`
- Branch: `main`

## Telegram-заявки

Форма бронирования умеет отправлять заявки в Telegram, если в Railway добавить переменные:

- `TELEGRAM_BOT_TOKEN` — токен бота от BotFather
- `TELEGRAM_CHAT_ID` — id чата, куда отправлять заявки

Если переменные не заданы, форма все равно работает на сайте и показывает подтверждение, но реальная доставка хозяйке будет отмечена как не подключенная.

## Страницы

- `/` — главная
- `/rooms` — комнаты
- `/host` — хозяйка
- `/location` — место
- `/booking` — бронирование
